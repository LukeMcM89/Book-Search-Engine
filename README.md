# Book-Search-Engine

A MERN stack application. Features a React front-end, Mongoose database and Node.js/Express.js server and API.

# Technologies Utilized / Required

* HTML
* CSS
* Javascript
* React
* Node.js
* Express.js
* Mongoose
* GraphQL
* Apollo Server
* Heroku
* Bootstrap styling engine
